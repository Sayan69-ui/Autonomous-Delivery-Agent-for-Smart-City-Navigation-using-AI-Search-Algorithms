# 🚚 Autonomous Delivery Agent (AI Project)

## 📌 Overview

This project simulates an autonomous delivery agent navigating a smart city grid using AI search algorithms such as BFS, UCS, and A*.

The system supports:

* Interactive map selection
* Dynamic terrain costs
* Algorithm comparison
* Real-time visualization
* Explanation of path decisions

---

## 🧠 Algorithms Used

* BFS (Breadth-First Search)
* UCS (Uniform Cost Search)
* A* Search (with Manhattan heuristic)

---

## 🗺️ Features

* User-defined start & goal
* Complex city map
* Traffic zones (high cost)
* Obstacle handling
* Path explanation panel
* Animated simulation

---

## ▶️ How to Run

```bash
python main.py
```

---

## 🖱️ Controls

* Left Click → Set Start & Goal
* Right Click → Add/remove obstacle
* Buttons → Run algorithms

---

## 📊 Output

Displays:

* Path
* Total cost
* Nodes explored
* Explanation of algorithm choice
